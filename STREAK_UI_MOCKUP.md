## Color Scheme

### Primary Colors
- **Green** (#10B981): Perfect day (100%)
- **Yellow** (#FBBF24): Good progress (75-99%)
- **Orange** (#F97316): Some progress (50-74%)
- **Gray** (#6B7280): Minimal (25-49%)
- **Light Gray** (#E5E7EB): None (0%)
- **Blue** (#3B82F6): Today indicator ring

### Gradient Backgrounds
- **Streak Card**: Orange-to-red gradient
- **Stats Cards**: White with shadow on hover
- **Motivational Cards**: Color-specific backgrounds
  - Yellow for streak break
  - Green for milestones
  - Purple for achievements
